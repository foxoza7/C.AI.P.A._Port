# Phase 03 - Docker Image

This phase demonstrates how the Python calculator application is containerized using Docker as part of the Cloud-Native AI Platform Architecture Portfolio.

## Repository

[Open Phase 03 - Docker Image](https://github.com/foxoza7/Docker_Image)

## What this phase demonstrates

Dockerfile creation
Docker image build process
Docker Compose execution
Python application containerization
GUI execution from a Docker container
Windows launcher integration
Basic Docker security and repository best practices

## Architecture Purpose

This phase represents the containerization layer of the portfolio.

The Python calculator application is packaged into a Docker image so it can be executed consistently across different environments such as local machines, CI/CD pipelines and future Kubernetes clusters.

Later phases will evolve this container into:

Docker image
→ GitHub Actions CI
→ Continuous Deployment
→ Kubernetes deployment
→ Monitoring and observability
→ AI platform workloads
